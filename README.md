# Minesweeper (Vanilla JavaScript)

A browser-based clone of the classic Minesweeper game, built with **HTML**, **CSS**, and **vanilla JavaScript**.  
Play directly in the browser with no frameworks or dependencies.

## Game page
https://ishaiperetz.github.io/Minesweeper/


## Features

- Classic Minesweeper gameplay with extra features.
- Left-click to reveal cells, right-click to place or remove flags.
- Automatic reveal of empty areas using recursion / flood fill.
- Timer and remaining mines counter.
- Configurable difficulty levels (e.g. Easy / Medium / Hard).
- Responsive layout that works on desktop and (optionally) mobile.


## Technologies

- HTML5 for structure.
- CSS3 for layout and styling.
- Vanilla JavaScript for game logic and DOM manipulation.
